# QoderWork 每日签到自动化

自动完成 QoderWork 桌面端的每日签到，领取积分。

## 环境信息

| 项目 | 值 |
|------|-----|
| 操作系统 | Windows 10/11 |
| 屏幕分辨率 | 2560 × 1440 (2K) |
| Python 版本 | 3.14+ |
| 依赖库 | pyautogui |

> **注意：** 脚本中的按钮坐标基于 2560×1440 分辨率测试。如果你使用其他分辨率或缩放比例，需要重新获取坐标并修改脚本配置。

## 文件说明

```
├── qoderwork_checkin.py    # 签到自动化主脚本
├── 管理签到任务.bat          # 定时任务管理工具（双击运行）
├── checkin.log             # 运行日志（自动生成）
└── README.md               # 本文件
```

## 安装依赖

```bash
pip install pyautogui
```

## 使用教程

### 1. 手动执行签到

确保 QoderWork 已打开，然后在终端运行：

```bash
# 正常签到（激活窗口 + 自动点击）
python qoderwork_checkin.py --max

# 测试模式（仅查找窗口，不执行点击，用于验证环境）
python qoderwork_checkin.py --dry-run
```

### 2. 设置定时任务（每天 9:00 自动签到）

双击打开 `管理签到任务.bat`，选择 `1` 创建定时任务。

创建后可在 Windows 任务计划程序中查看 `QoderWork_DailyCheckin` 任务。

### 3. 管理定时任务

双击 `管理签到任务.bat`，按提示选择：

| 选项 | 功能 |
|------|------|
| 1 | 创建定时任务（每天 9:00） |
| 2 | 删除定时任务 |
| 3 | 查看任务状态 |
| 4 | 立即手动执行一次签到 |
| 5 | 测试模式运行（不点击） |

## 适配其他分辨率

如果更换了显示器或调整了分辨率/缩放，需要更新脚本顶部的两个坐标：

```python
# qoderwork_checkin.py 配置区域

CHECKIN_ENTRY_BUTTON = (2218, 19)    # "签到/邀请，赚积分" 按钮
CHECKIN_CONFIRM_BUTTON = (2204, 270)  # "签到" 确认按钮
```

**获取坐标方法：**

1. 打开 QoderWork，全屏显示
2. 打开系统自带的「截图工具」或使用 PowerToys 的「屏幕标尺」
3. 将鼠标悬停在目标按钮上，记录屏幕坐标
4. 修改脚本中对应的坐标值

## 注意事项

- 定时任务触发时，QoderWork 需要处于**已启动**状态（不需要在前台）
- 脚本会自动重试查找窗口，最多 5 次，每次间隔 3 秒
- 鼠标快速移到屏幕**左上角**可紧急中断脚本（pyautogui 安全机制）
- 签到结果可查看同目录下的 `checkin.log` 日志文件

## 原理

脚本通过 Win32 API 枚举并激活 QoderWork 窗口，再使用 pyautogui 按预设坐标模拟鼠标点击，完成签到流程。整个过程约 7 秒。
